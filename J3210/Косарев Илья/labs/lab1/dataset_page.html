cocodataset / COCO-2017

Широкомасштабный датасет для обнаружения объектов, сегментации и описания изображений.

Создатель
cocodataset
Image Search CSV Object Detection CC BY 4.0

Описание датасета

COCO 2017 (Common Objects in Context) — это масштабный датасет для задач обнаружения объектов, сегментации изображений и распознавания ключевых точек. Он содержит более 330 тысяч изображений, на которых детально размечено 1.5 миллиона объектов в 80 различных категориях (люди, транспорт, животные и др.).

Пример использования

Вы можете легко загрузить аннотации датасета с помощью библиотеки pandas для просмотра метаданных изображений.

import pandas as pd
import json

# Загружаем JSON аннотации
with open("annotations/instances_train2017.json", "r") as f:
    coco_data = json.load(f)

# Конвертируем список изображений в DataFrame
df = pd.DataFrame(coco_data["images"])
print(df.head())
