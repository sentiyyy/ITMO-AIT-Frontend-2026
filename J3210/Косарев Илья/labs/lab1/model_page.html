ultralytics / YOLOv8-Detection

Мощная модель для детекции объектов в реальном времени.

Создатель
ultralytics
Computer Vision PyTorch Object Detection GPL-3.0

Описание модели

YOLOv8 (You Only Look Once version 8) — это новейшая версия семейства моделей YOLO, созданная Ultralytics. Она устанавливает новый уровень state-of-the-art для задач детекции объектов, сегментации изображений и классификации. Модель оптимизирована для работы на CPU и GPU.

Пример использования

Вы можете легко запустить модель на Python, используя встроенный интерфейс библиотеки ultralytics.

from ultralytics import YOLO

# Загружаем предобученную модель
model = YOLO("yolov8n.pt")

# Запускаем предсказание на картинке
results = model.predict("https://ultralytics.com/images/bus.jpg")
